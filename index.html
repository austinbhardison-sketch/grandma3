Sequences, Executors, Presets & Groups
Overwrite vs. Merge ▼
OptionWhat it does
OverwriteReplaces the existing look entirely with the new look.
MergeAdds the new look on top of the existing — both coexist in the cue.
Move Black ▼

Allows fixtures to prepare for an upcoming cue during a blackout. Color, gobo, and position changes happen while the light is off. Especially useful in theatrical settings.

Access: Sequence Sheet → top-right settings toggle → Move Black column.

ModeBehavior
NoneFixtures never prepare during dark moments.
EarlyFixtures prepare as soon as they go dark.
LateFixtures wait until the last possible dark moment.
Note: Cyan cell highlight in track sheet = fixture is preparing. Requires a blackout cue in between. Early and Late behave identically if only one blackout cue exists. Can be set globally or per individual cue.

Executor Button Functions ▼
FunctionBehaviorFaderOff Cue
Go+Advance sequence——
FlashOn while heldForces 100%No
ToggleTap on / off——

Flash vs. Temp: Flash forces 100% and skips the off cue. Temp plays at the current fader level and triggers the off cue on release — so fade-out timing works correctly.
Fader Types ▼
TypeBehavior
Master (MST)Puts the sequence on. Doesn't consider playback position.
Temp FaderCross-fades the cue on as fader rises, off as it lowers. Great for catwalk-style looks.
Preset Modes ▼
ModeApplies ToDefault For
Selective SOnly fixtures used to store it.Position
GlobalAll fixtures with the same fixture profile/mode.Color
UniversalAll fixtures that have the attribute at all.Dimmer
Note: To manually change preset mode before storing: long press Store → change Preset Mode at top right → tap pool. To change a pool's default: tap the MA logo in the preset pool header.
Best practice for color presets: Select one fixture of each type (101 + 201 + 301 + 401), color-correct each visually, then Store/Merge all into one global preset. Avoids Universal's color mismatch problem.
Update a Preset ▼

Nudge any attribute to activate it → tap Update → tap the navy blue line with the preset name. Auto-saves and exits.

update preset [number]
Align Tool ▼

Hard key between the two command screens, or soft key in the encoder bank. Tap once to activate. Deactivates as soon as you touch a second encoder — re-tap to reactivate.

Mode (tap to cycle)Behavior
Align / GoFan symmetrically from center
Arrow leftLeft fixture fixed, right moves freely
Arrow rightRight fixture fixed, left moves freely
Arrows inward →←Outermost move freely, innermost stay
Arrows outward ←→Innermost move freely, outermost stay
Color tip: Works with color too — open the CIE gradient tab, tap Align, drag horizontally for a color spread. Mirrored arrow mode = color "wings" effect.
All Pool ▼

Stores every currently active attribute with no filtering — dimmer, color, position, all at once. Found underneath the color picker in the special dialog window on the first screen.

Tick-mark bar at the bottom shows which feature groups are stored. Gray bar under S = dimmer level.

Program Time ▼

Far right of encoder bar next to MA tricks overlay. Tap "Prog Time" label at top of fader to enable. All preset taps and dimmer changes fade to the value over the set time instead of snapping. Good for busking.

Cues and Sequences ▼
ConceptExplanation
Sequences live in the poolNot on the executor. Delete the executor and the sequence still exists. Executor is just a handle.
Select a sequenceTap Select key → tap its executor. Yellow box = selected in pool.
Store to selected sequenceOnce selected, Store → Please stores directly without tapping the executor again.
Playback values in fixture sheetValues in yellow with a sine-wave underline = coming from playback.
Trigger Types ▼
TypeBehavior
GoOnly advances when you manually press Go.
FollowAfter cue finishes, counts down trigger time, then auto-fires next cue.
TimeAs soon as Go is pressed, countdown starts immediately — doesn't wait for current cue to finish.
Delay vs. trigger time: Trigger time controls WHEN the cue fires. Delay controls how long after firing before the fade begins.
Timing Syntax ▼

Set fade and delay on a cue from the command line:

time 3 time 1 cue 3
→ 3-second fade, 1-second delay on cue 3
Store Pop-up Options ▼
OptionWhat it does
OverwriteDiscards all existing cue data and replaces with new.
MergeKeeps existing cue data and adds new values at higher priority.
RemoveRemoves active attributes from cue — back to default, no values stored.
ReleaseGives those attributes a release value so another cue at the same priority can take control.
Create cueAdds a new cue to the sequence.
Remove vs. Release: Remove kills the attribute entirely. Release lets it exist at the value but hands priority to other sequences. Easy to accidentally hit the wrong one.
Tracking & Track Sheet Colors ▼

Tracking = a fixture stays in whatever state it was last told until explicitly changed. Values track forward through the sequence until overwritten.

ColorMeaning
CyanStored in this cue (dimmer going up)
MagentaTracked from a previous cue — NOT stored here
GreenStored in this cue (dimmer going down)
WhiteStored at the same value as the previous cue
Queue Only: Choose from the pop-up when storing to keep a blackout contained to just that cue. If you accidentally track a blackout: Update → expand options in yellow header → change to "Add New Content" → tap navy line.
Copy Cue Modes ▼
copy cue 3 at cue 6 please → choose Content or Status
ModeWhat it copies
ContentOnly values actually stored in the source cue. Tracked values NOT included.
StatusValues stored PLUS everything tracked into the cue — copies the full look.
IF Syntax ▼

Selects everything currently outputting — then put those at zero and store a blackout that only affects what's on:

if → please
Info Window / Notes ▼

Open from the More tab (after Help). Set "Link Type" to Selected Sequence and "Window Mode" to Current Queue. Notes from the cue's notes column display in real time as you advance through the sequence.

Find the Notes column by scrolling all the way right in the sequence sheet (two-finger scroll).

At Copy — Fixture at Fixture ▼

Copies values from one set of fixtures to another. The word "fixture" before the source is required — without it the console assumes a dimmer value.

fixture 113 through 124 at fixture 101 through 112 please

Mirror — reverse the source range to flip the look:

fixture 107 through 112 at fixture 106 through 101 please
Self Fix Key ▼
ActionResult
Self Fix → tap presetSelects all fixtures used to store that preset, then applies it.
Self Fix → tap cue/sequenceSelects all fixtures used in that cue/sequence.
Double-tap Self FixPuts Shuffle in command line — tap a group to shuffle fixture order.
Warning: Do NOT double-tap a group. This activates ALL attributes including control channels and stores zeros everywhere — silently wrecks tracking throughout your show.
Groups — Storing ▼

Select fixtures → Store → tap group pool object.

Swipey gesture: tap and immediately drag (no hold) on the pool object → slide to Store → release. Same gesture works for Label.

Appearances pool (right of groups view): swipe on a color appearance → release on Assign → tap the group.

Grid Groups — Blue Box Method ▼

Blue box shows where the next fixture will land in the selection grid. Tap to move it, then type the next fixture number.

Setup toggle (top right of selection grid) lets you tap and drag fixtures to rearrange after placing.

Grid Keyword ▼

Creates a grid of empty slots. X key 3 puts "grid" in the command line. Format is X/Y (or X/Y/Z for 3D).

grid 0/0 through 19/19 please
→ creates 20×20 grid
401 through 800 please
→ fills the grid with those fixtures

MA tricks Overlay ▼

Opened from the encoder bar. Reset button (top right) clears all settings without clearing your fixture selection.

ToolWhat it does
X blocksGroups fixtures into paired grid points (e.g., X blocks 2 = groups of 2).
X wingsFans from outside in — outermost share a point, works toward center.
X groupsSeparates evens/odds (or thirds, quarters…) into distinct grid points.
X shuffleShuffles fixture order in selection grid only — does NOT change output.
X shiftCircular shift — loops fixtures through the selection (MA2 "circular copy").
Mirror (transform)Makes the phaser symmetrical — not the same as X wings.
MA tricks pool: Store a configuration as a pool object so you don't have to re-open the overlay each time. Tap to apply. MA tricks stack — they don't override each other. Always reset between applying different ones.
Pre-defined MA tricks: Menu → Show Creator → MA tricks → select all → Import.
Quick gradient shuffle: set gradient (at 0 through 100) → open MA tricks → Copy Selection → Shuffle → Copy Values.

Software Versions ▼

Format: Major.Minor.Streaming.BugFix. Cannot load a show saved in a newer version on an older console. Can load older files into newer software.

Do not update mid-show-run. Even a bug fix version can break something you relied on. Migrate very old files stepwise through each version — don't jump from 1.4 to 2.3 directly.
Saving & Loading Shows ▼

USB file path: grandMA3/shared/shows

Keep existing DMX/network config: Uncheck DMX Ports and DMX Protocols when loading. It keeps what you already have — does NOT revert to default.

Networking, Phasers, Recipes, Magic & Layouts
Networking Basics ▼

An IP address is made of four octets (0–255 each). Think: octet 1 = state, 2 = town, 3 = street, 4 = house. The subnet mask defines which addresses can talk to each other — 255 locks that octet, 0 allows it to change.

Subnet MaskWhat can changeRange
255.0.0.0Last three octetsBroadest
255.255.0.0Last two octetsMedium
255.255.255.0Last octet onlyTightest — max 254 devices
ArtNet note: Defaults to 2.x.x.x range — cannot have a gateway, causes conflicts with internet traffic. Default to sACN when possible.
Network Menu ▼

Menu → Network. My Interfaces button (top right): edit IP addresses and subnet masks per physical port.

Warning: Hard keys do NOT work while the My Interfaces window is open — it locks the rest of the console. Set DHCP to No before editing IP. Press Apply Changes when done.

MA net interface (bottom): selects which physical port MA sends out of. Cannot be changed while in session. Tap the big red power button to end/create a session.

Session Status Types ▼
StatusIndicatorMeaning
StandaloneRed treeCompletely off network. Blocks ALL protocols.
Idle MasterGray treeIn session alone — others can join. Must be here to send/receive any protocol.
Global MasterBlue treeMaster of a multi-desk session.
ConnectedGreen treeInvited into and in session with a global master.
sACN / ArtNet Settings ▼

Menu → DMX Protocols → choose sACN or ArtNet from left side.

SettingWhat it does
Interface toggleChooses which physical port (con 1/2/3) to send out of.
Enable OutputMust be ON to send sACN.
Send sACN if Idle MasterMust be ON when running a show without other MA desks in session.
Local UniverseWhere fixtures are patched inside MA — the starting universe in MA. Not an amount.
sACN UniverseWhat universe the external node is expecting data on.

Local start example: If fixtures are patched starting at universe 5 in MA, local start = 5. If your first four universes are house lights on physical DMX ports, sACN rig starts at universe 5.
Patch vs. Live Patch ▼
ModeChanges allowedResync?
PatchAdd/delete fixtures, change types, all major changes.Yes — interrupts network users briefly.
Live PatchRename, change DMX addresses, invert pan/tilt only.No — safe in multi-user environments.
Patching New Fixtures ▼

Menu → Patch → tap "new fixture" at the bottom of the list → Insert New Fixture → Library tab.

Library iconWhat it shows
MA logoInternal library — factory profiles.
Person iconUser-created profiles stored on internal drive.
Globe (grayed)GDTf Share world server — requires internet.

After selecting a fixture type, set: Mode, Quantity, Fixture ID (FID), and Patch address (e.g., 1.001 or 1.1).

Parameters vs. DMX channels: Parameters = attributes (pan, tilt, dimmer = 3 parameters). DMX channels account for bit depth (pan + pan fine = 2 channels but 1 parameter). MA3 output licensing is based on parameters, not DMX channels.
Grouping Fixtures (Patch Folders) ▼

Folder containers in patch — not groups in the programming sense. Found in show library under Generic → Grouping. No DMX address needed.

To add fixtures: cut (not copy) fixtures from patch list → tap "new fixture" cell inside grouping fixture → paste → choose Absolute.

Best practice: Use groups for programming, use grouping fixtures for patch organization only.
Park ▼

Pauses output of specific DMX addresses exactly where they are. Cannot be overridden by cues or the programmer until unparked.

ActionHow
Get Park in command lineDouble-tap small Pause key (under X key 9)
Get Unpark in command lineDouble-tap small Go+ button (under X key 11)
Park a full fixturepark 101 please
Park a specific attributepark → tap feature group on encoder bank
Unpark entire universeunpark → hold MA + tap X8 → through please
Real world uses: House lights at a set level during tech, parking relays open/closed, a work light for notes, a practical that must stay on.
Phasers — Overview ▼

A phaser is a multi-step preset. Every preset starts on step 1 — a phaser continues from there. Create a step by pressing the right arrow (→) in the step bar on the encoder bar.

Encoder bar layerWhat it controls
ValuesAttribute values per step
TimingFade/delay per step
Phaser OverallApplies to entire loop — speed and phase. Cannot be per-step.
Phaser StepsPer-step: acceleration, deceleration, transition, width.
Width vs. Speed: Speed (Phaser Overall) applies to the whole loop. Width (Phaser Steps) controls how long/short a specific step is — use width when you need different timing per step. Transition = 0 means instant snap.
Phaser Window & Toolbar ▼
ViewBest for
1DSeeing the waveform shape
2DAerial grid — best for building position phasers
SheetTable of all values per step — see which presets are in which step
Toolbar toolWhat it does
A+ toolAdds an absolute step — tap in 2D grid to place a point
R+ toolAdds a relative step
Move toolMoves existing points in 2D grid
Lightning boltPredefined forms: sine, ramp, square, circle, etc.
Phase tool (top left)Sets phase spread — e.g., choose 360 to spread across the loop
Size tool (bottom left)Resize the shape — use with All Steps selected
All Steps toggleMakes toolbar actions affect all steps simultaneously
Phase (0–360): Without phase, all fixtures do the same thing at once. With 360°, fixtures are evenly spread across the loop — creating a chase.
Absolute vs. Relative Phasers ▼
TypeBehaviorWhat stops it
AbsoluteBuilt on the absolute layer — defines exact values.Another absolute preset or phaser on the same attribute.
RelativeBuilt on the relative layer — runs relative to whatever absolute anchor is applied.Only stopped by a relative static preset or Stomp.
Why this matters: Apply an absolute position preset (the anchor) and a relative circle phaser (the movement) — the circle moves with each new position preset without stopping. This is what makes the singer/guitarist/bassist workflow possible.
Stomp ▼

Knocks in the last outputted single-step value from any layer, stopping a phaser and holding fixtures at their last static position. Stomp key is between the two command screens.

stomp → tap position encoder bank
→ stops position phaser, holds last position
Single Step Mode ▼

Toggle in the encoder bar (left side near Align). Pauses the phaser's stepping so you can see and edit each step individually. Use left/right arrows to navigate. Tap again to disable.

X5 key color phaser shortcut: Hold X5 and tap multiple color presets in sequence — automatically creates a multi-step color phaser. Only works with presets, not the raw color picker.

Integrate command: When in step 1 of an existing phaser, tapping a preset normally overwrites it. To add to step 1 instead: hold MA + tap At → Integrate in command line → tap the preset.

Recipes — Overview ▼

A recipe has three ingredients: Selection (which fixtures — a group), Value (what value — a preset), and MA tricks (how to apply the values).

Recipes are a different workflow, not a different feature — they can live in presets or in cues. They allow you to update an entire look instantly by changing just the selection or value.

Recipe Preset — Without Selection ▼

Acts like a template effect from MA2. Any group you apply it to gets the effect.

How to make: Store a phaser → swipe on empty preset in pool → Edit Setting → Turn Into Recipe → leave selection empty → edit value cell (choose the phaser) → set MA tricks (e.g., phase X = 360) → label → escape.

Apply by tapping the recipe with any group selected.

Recipe Preset — With Selection ▼

Always applies to the specific group stored in it — not the currently selected fixtures.

How to make: Create a group → swipe on empty preset → Edit Setting → Turn Into Recipe → set selection to your group → set value (preset) → set MA tricks → escape. Double-tap to apply.

Key benefit: Add more fixtures to the group later and the recipe automatically includes them — great for touring where rig size changes per venue.
Recipes in Cues ▼

In the sequence sheet: tap MA logo → Mask tab → enable Show Recipes. Recipe lines appear at the bottom. Long press "New Recipe" cell to add a line.

Set selection (group) and value (preset) in the cells. Set Phase to X in the far-right cells.

Quick update trick: Lasso multiple selection cells → change to a new group — updates all recipe lines at once. Much faster than rebuilding a cue.
Cook ▼

Used when conflicting values exist — e.g., a hard programmer value says 100%, a recipe line says 50%. Cooking takes the recipe value.

A green cooking pot icon appears on any preset or cue that has been cooked.

Magic Presets ▼

Define specific endpoints (A, B, and optionally midpoints) and the console dynamically spreads all selected fixtures across those points. Identified by a green wizard hat icon.

TypeHow to make
2-point magicSelect 2 fixtures → set each to different values → tap Set → Store as preset → swipe → Edit Setting → tap Magic toggle (top left) → escape.
3-point magicSame with 3 fixtures — middle point determines arc height.
Best practice workflow: Use magic to build a position look (e.g., straight apron, cross, fan), then store those positions as regular selective presets. Magic builds them, presets recall them.
Fixture-type specific: A separate magic preset is needed for each fixture type.
Layouts ▼

A 2D sandbox to freely arrange fixtures, groups, presets, sequences, and other objects. In MA3, objects are assigned to layouts (not stored like in MA2).

ActionHow
Assign fixturesSelect fixtures → tap Assign key → tap anywhere in the gray canvas.
Move objectsEnable Setup (top right) → lasso → tap and drag.
Auto-place from 3DSetup enabled → encoder bar → Arrangement → Layout Type → Camera → scroll to "2D Top" → Apply.

Set action on elementEdit → tap object → set action (Go+, Flash, Toggle, etc.).

MA Start Show ▼

Built-in busking show file included with every MA3 installation. Everything is built using recipes and macros. Good for last-minute gigs or as a base template.

Load via: double-tap the orange macro (5001) → follow the prompts to store groups and position presets.

Group numberWhat to store
1, 2, 3Gridded groups — one per fixture type (X width = half fixture count)
19, 20, 21Linear groups — one per fixture type
37, 38, 39Odd-numbered fixtures (X groups = 2 → tap Next)
55, 56, 57Even-numbered fixtures (X groups = 2 → tap Next again)
Always merge when storing to Start Show groups — never overwrite. The groups are already referenced in recipes, and overwriting breaks those references.

LOS layout (bottom of start show view bar): pre-built layout for X key hit buttons. Select group → set dimmer, position, color, fade and delay → tap and hold X3 / X4 to assign.

Partial Show Read (PSR) ▼

Found at the bottom left of the Load menu. Brings specific elements from one show file into another — much more targeted than a full show load. Best way to import your patch into a fresh show like the Start Show.
